1. Summary

当前示例聚焦于典型开源库形态：核心代码位于根目录 `src/`， Rust/WASM 位于 `wasm/`，示例页面位于 `examples/`，并且通过 `dist/dompdf.js` 直接运行，不再依赖 Vite playground。

这个页面混合了图片、表格、渐变、透明度、长列表、强制分页与 `divisionDisable`，方便直接观察导出的 PDF 是否与 HTML 接近。

Use Case

  • 验证打包后的 UMD 产物能直接在浏览器中使用。
  • 作为 README / examples 中的基础接受测试页面。
  • 展示中文字体、分页和视觉样式的综合能力。
